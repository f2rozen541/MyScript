Graphics

  • Graphics

    • MyScript Graphics is designed for building applications that enable the recognition of hand drawn graphics created with digital ink and real-time conversion into beautified graphics. It is ideal for creating organization charts, custom sketching, diagram-based and note-taking applications through real-time or asynchronous stroke processing.

    Graphics input use case is part of the Application Toolkit (ATK) and the Cloud Development Kit (CDK) which come with a suite of features including:

    • ink capture,
    • ink rendering,
    • real-time recognition,
    • editing gestures.

    So developers can provide users a natural handwriting experience through careful consideration of interaction and behavior.
