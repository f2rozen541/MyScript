MyScript Application Toolkit (ATK)

This documentation aims at helping you integrate our MyScript widgets and components into your applications. It is intended for developers with good knowledge of object-oriented programming.

For more information on Text, Math, Graphics, click on the corresponding tab in the top-right menu.

Certificate

A certificate is required to use the MyScript handwriting recognition engine and gives the right to use the ATK. Each certificate gives access to all widgets for a given platform.

You can download it from the Support website.

Sample Execution

To integrate MyScript widgets and components on Android, you must be using the latest version of Android Studio and have all Android tools and SDK (level 15 or up) installed and configured properly.

  1. Unzip MyScript_ATK_OEM-android-*.zip to obtain this tree:

    • docs - Contains the whole documentation (installation, functional, API).
    • libs - Contains libraries.
    • samples - Contains sample applications, showing basic use cases.
    • third_party - Contains ATK and third-party licenses.
    • atk-info.json - Contains information about the version numbers.

  2. Copy the downloaded certificate into samples/myscript-certificate/src/main/java/com/myscript/certificate/MyCertificate.java to replace the existing one.
    From now on, samples can be built and launched.

  3. In Android Studio, click Open an existing Android Studio project and browse towards the samples folder of your package.

  4. Build and run your sample.

For more information on Gradle builds, please refer to the Gradle documentation.
